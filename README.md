# portal-macaujc-journal

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能包含静态信息、工具界面或其他轻量级前端内容，适合通过 GitHub Pages 直接访问。

仓库本身不绑定任何特定域名或网站，仅为 HTML 文件的集中存储与版本管理提供空间。

## 目录说明

```
.
├── index.html          # 首页入口文件（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要存放独立 HTML 页面，每个文件可视为一个独立功能或内容单元。
- `assets/`：页面引用的公共资源，按类型分目录存放。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，无强制跳转或依赖关系。
- 归档内容可能不定期更新，建议以仓库最新 commit 为准。
- 如需引用页面，请使用 GitHub Pages 生成的原始 URL 或直接下载文件。

## 维护说明
